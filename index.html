A scalable web search engine

Dec 3, 2023

This project implements a search engine with an inverted index built using MapReduce. It has an Index Server that provides search results via a REST API by combining TF-IDF and PageRank scoring. It also includes a Search Server web application that queries the Index Server and displays the top ranked results to users.

Instagram clone

Oct 15, 2023

The project involves building an Instagram clone called Insta485 with a client-side React.js front-end that displays a dynamic feed of posts, comments, and likes by fetching data from a custom-built REST API back-end using Python/Flask. It covers concepts like component rendering, state management, API integration, and end-to-end testing while reusing some server-side code from a previous project for authentication and user pages.
