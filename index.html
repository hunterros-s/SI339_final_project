Simple map reduce implementation

Nov 5, 2023

Implemented a MapReduce framework consisting of a Manager and multiple Worker processes communicating over TCP/UDP sockets using a JSON protocol. The Manager partitions input data, assigns map and reduce tasks to Workers, and handles fault tolerance. Workers execute the assigned map/reduce tasks by running provided executables and partitioning/sorting/merging intermediate data using disk-based algorithms to minimize memory usage.
